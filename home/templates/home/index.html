Frequently asked questions

Questions asked frequently and a small guideline for everyone.

Any authority can create an account and start using it. The application is open to all. Authority have to entry applications, that will give a unique application id. This will be attached to the application. The entry is also mapped with the email id provided by the applicant. The authority can update the status of the application anytime. The application then later can be searched by the application id or the email id.

This is completely free to use. In fact the whole project is open source and can be expanded by anyone with more exciting features.

If searched with email, an otp will be sent on the email. The otp will be valid for 5 minutes. The otp will be required to view the applications.
